Ground State (GS) properties are taken from Nubase2012, and separation values from AME2012 link

For multiple 0-energy isomers, the degeneracy is removed with the following cascade of options:

  1. If there is a clear match with a nubase NON-0 energy level, this energy is used
  2. If there is a clear match with a nubase 0-energy level, the uncertainty of this level is used as level energy
  3. Default: the level is assigned an energy of 0.1 keV + the energy of the previous level
All other levels for which the energy cannot be determined ( X + value, where X > 0 ) are removed.
All levels having the same energy are assigned a splitting of 0.1 keV.

If between two consecutive levels there is an energy gap > 8 MeV AND > 4.5 * 30 / A0.67 MeV, all the levels above the gap are removed.

The internal conversion factor are calculated using BRICC

The FORTRAN format of the level record has two new fields at the end:
(i3,1x,f10.6,1x,f5.1,i3,1x,(e10.2),i3,1x,a1,1x,a4,1x,a18,i3,10(1x,a2,1x,f10.4,1x,a7),f10.6,1x,3(2i))

  • The energy-shift applied
  • The band(s) to which the level is member
